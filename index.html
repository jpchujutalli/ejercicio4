Ramas de programacion

Todas las ramas de la programación tienen algo en común: el pensamiento lógico y el uso de algoritmos. Pero, como veremos a continuación, cada una de ellas se aplica a un nicho empresarial diferente y utiliza metodologías y lenguajes de programación que le son propios. Una vez que conozcas las distintas variantes que hay dentro de la programación, podrás elegir el camino más adecuado para ti.

1. Programacion web

Como programador web, te podrás dedicar a escribir los códigos para crear distintas aplicaciones que se utilizan en internet. Una de las tareas más demandadas en este tipo de programación es la creación de páginas web. Algunos de los lenguajes que se utilizan en programación web son:

PHP

JSP

ASP

RUBY

PYTHON

NODE

HTML

JAVASCRIPT

2. Programación desktop

Este tipo de programación se especializa en la escritura de códigos para las aplicaciones que se utilizan en las computadoras, por ejemplo, los procesadores de texto que tienes instalados en tu laptop o computadora de escritorio, hojas de cálculo, programas de reproducción de audio, aplicaciones de la cámara web, calculadora, programas para edición de imágenes, entre muchos otros. Algunos de los lenguajes más comunes que usan este tipo de programadores son:

C++

C#

JAVA

PYTHON

3. Programación de aplicaciones

Todas las aplicaciones que tienes en tu dispositivo móvil, han sido desarrolladas por programadores, ya sea para los sistemas Windows Phone, iOS o Android. Por ejemplo, todas las versiones móviles de las aplicaciones desktop como Instagram y Facebook, son adaptadas por programadores de aplicaciones. Pueden usar uno o más de los siguientes lenguajes:

C++

JAVA

Objective-C

4. Programación backend

Esta es la rama a la que pertenecen los programadores que desarrollan el soporte o infraestructura de las aplicaciones, por ejemplo el manejo del servidor y la base de datos. Estos programadores pueden crear aplicaciones desde cero y emplean lenguajes como:

PYTHON

RUBY

JAVA

PHP

5. Programación frontend

Al contrario de la programación backend, esta rama de la programación se dedica a trabajar la parte de la aplicación que está frente al usuario. Un programador de frontend se especializa en el envío y recepción de datos en esta fase de la programación y asegura que la comunicación con el usuario sea precisa y eficaz. Para lograrlo, utilizan alguno de los siguientes lenguajes:

HTML

JavaScript

CSS

JSON

XML

API

6. Programación fullstack

Esta rama de la programación es una combinación de backend y frontend, por lo que es una especialización más completa. Si llegas a dominar la programación fullstack, podrás ampliar tus posibilidades laborales, ya que serás capaz de resolver una amplia gama de problemas asociados con el desarrollo de programas y aplicaciones. Sin embargo, al dedicarte al fullstack corres el riesgo de no dominar ninguna de las dos ramas por completo, lo que podría limitarte en algunos casos.

7. Programación de videojuegos

Si te interesa incursionar en el complejo mundo de los videojuegos y estudiar para ser programador de videojuegos, debes saber que la programación es solo una pequeña parte en el desarrollo de esta tecnología. El resto del equipo está conformado por diseñadores, animadores, músicos y escritores. El lenguaje de programación más utilizado para este fin es el C#, el cual da vida al motor de videojuegos conocido como Unity 3D.
